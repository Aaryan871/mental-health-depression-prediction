# 🧠 Mental Health Depression Prediction

An end-to-end Machine Learning project that predicts whether a person is at risk of depression based on mental health and lifestyle-related features. This project includes data preprocessing, exploratory data analysis (EDA), feature engineering, model training, evaluation, and deployment using Streamlit.

---

## 📌 Project Overview

Mental health is an important aspect of overall well-being. Early prediction of depression can help individuals seek timely support and treatment.

This project uses Machine Learning algorithms to classify whether a person is likely to experience depression based on various input features.

## 💻 Installation

Clone the repository

```bash
git clone https://github.com/Aaryan871/mental-health-depression-prediction.git
```

Go to project folder

```bash
cd mental-health-depression-prediction
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the Streamlit application

```bash
streamlit run app.py
```
